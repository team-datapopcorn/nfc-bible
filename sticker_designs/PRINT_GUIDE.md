## 인쇄 사양

| 항목 | 설정 |
|---|---|
| 인쇄 방식 | 도무송형 (자유형 칼선) |
| 용지 | 아트지 (유광) 또는 모조지 (무광) |
| 코팅 | 유광 라미네이팅 권장 (내구성) |
| 해상도 | 300dpi 이상 |
| 색상모드 | CMYK |
| 여백 | 칼선 안쪽 2mm 여유 |
| 재단선 | 칼선 바깥 3mm 도련 |

## NFC 칩 부착 위치

```
┌──────────────┐
│              │
│   ┌──────┐   │  ← 스티커 뒷면 중앙에
│   │ NFC  │   │     NTAG213 칩 부착
│   │ 25mm │   │     (원형 스티커형)
│   └──────┘   │
│              │
└──────────────┘
```

- NTAG213 스티커형 (25mm 원형) 사용
- 스티커 후면 중앙에 부착
- 칩 위에 스티커 인쇄면이 오도록 배치

## 주문 시 전달 사항

```
- 도무송형 자유형 스티커
- 디자인별 100매씩 (총 300매)
- 용지: 아트지 유광 / 모조지 무광 (택 1)
- 코팅: 유광 라미네이팅
- 칼선 제작 (AI 파일 별도 제공 또는 업체 제작 요청)
- CMYK 300dpi
```
